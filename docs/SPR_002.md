# SPR-002 — Toolbar

**Sprint Goal:** Deliver a functional toolbar integrated into the application window, providing quick access to viewport controls and laying the groundwork for future tool modes.

**Milestone:** Toolbar  
**Status:** Complete

---

## Background

With the 3D viewport established, the next step is giving users a proper toolbar. This provides a visual home for mode switching (e.g. select, sketch) and viewport utilities (e.g. reset camera, toggle wireframe). The toolbar should integrate cleanly into the custom OpenGL UI layer alongside the viewport.

---

## Technical Notes

- **Toolbar widget:** Draw buttons and menus directly in the custom UI renderer for full control over look and interaction
- **Radio behaviour:** Keep Select / Sketch exclusivity in custom UI state
- **Wireframe mode:** Renderer now swaps between solid cube geometry and explicit line geometry
- **Icons:** Text-first controls are acceptable until a custom icon atlas exists
